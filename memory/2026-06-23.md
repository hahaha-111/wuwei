### 扫描 API 签名（官方文档 2026-06-12）
- `scan.startScan(filters: Array<ScanFilters>, options?: ScanOptions): Promise<void>`
- `scan.stopScan(): Promise<void>`
- `scan.on('deviceFound', callback: Callback<Array<ScanResults>>): void`
- `scan.off('deviceFound', callback): void`
- `ScanResults` 字段：`address`, `deviceName`, `rssi`, `data` (ArrayBuffer)
- 广播数据类型常量：`SHORTENED_LOCAL_NAME = 0x0A`, `COMPLETE_LOCAL_NAME = 0x0B`（与 BLE 不同！）
- `BusinessError` 从 `@kit.BasicServicesKit` 导入
- 扫描过滤器至少需要一个条件，用 `[{}]` 作为通配（不能用 `[]`）

### ArkTS 编译限制
- catch 从句不能用类型标注：`catch (err)` ✅ / `catch (err: Error)` ❌
- Promise `.catch((err: BusinessError) => ...)` 允许类型标注
